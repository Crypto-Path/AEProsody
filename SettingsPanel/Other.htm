Help:

Starting:

To start press the SPACE button on your keyboard. If done correctly notes (Object/Images, depending on the skin, likely have a diamond shaped with a futuristic tech sprite) will start falling.

Hitting your first note:

When a note reaches one of the note hitters (The 4 static non-moving objects that looks similar to the note but are lacking in detail comparatively) you press the respective key set in the key bindings
1 - Far left
2 - Middle left
3 - Middle rightt
4 - Far right
if you don't know what keys the number refer to you can use detect. Click detect, then press the key you want to assign that hitter to.
Default Values:
1 - 68:D
2 - 70:F
3 - 74:J
4 - 75:K.
